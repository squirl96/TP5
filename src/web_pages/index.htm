Générateur de fonctions ETML-ES
 

Bienvenue!

Vous pouvez régler ci-dessous les paramètres du générateur de fonctions, puis valider en appuyant sur OK.

Forme :      
Fréquence : ~freq~ 20 2000
Amplitude : ~ampl~ 100 10000
Offset : ~offset~ -5000 +5000

Caractéristiques du serveur

Stack Version: ~version~
Build Date: ~builddate~
File System Location: ~drive~
File System Type: ~fstype~
